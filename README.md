# 💰 Treasury Cash Flow Project

Streamlit app to analyze and predict treasury cash flow using machine learning.

## Features
- Cash flow prediction
- Interactive dashboard
- Downloadable reports

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

## Run App
streamlit run app.py
